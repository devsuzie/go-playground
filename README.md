## command
- `go mod init`
  - go mod 파일 생성 
- `go run .`
  - go 파일 실행 명령어
- `go mod tidy`
  - 최신 패키지로 설치
  - 패키지를 신기하게 가져오네.. package / symbol 찾는 곳은 [여기](https://pkg.go.dev/)
